D&B API -- Get Requests Overview

Interfaces with D&B Direct 2.0 API.
Key to most of the API calls is a D-U-N-S number
A D-U-N-S Number is a unique 9-digit number identifying a business on which a D&B Credit Report has been generated; and once assigned, is neither reused nor assigned to another business.

Query Parameters

Param Description
qtype Query Type: Name of D&B Operation To Invoke Below listed is a table of qtypes. Each of these qtypes represent the different D&B Operations / API's that can be invoked
qparam Query Parameter: Parameter to be passed to the D&B Operation

Query Parameters

QType Description QParam Description
search Performs a Keyword based search for a company name. Invokes the FindCompany API keyword String. Can contain space, api takes care of encoding the special characters.
findIndustry Performs a Keyword based search for a industry name Invokes the FindIndustry API keyword String. Can contain space, api takes care of encoding the special characters.
profile Gets Company Information For A D-U-N-S number duns string up to 15 chars. Pattern: {([0-9a-zA-Z]*)}
competitors Gets a List Of Competitors For A D-U-N-S number. Invokes the Find Competitors API duns string up to 15 chars. Pattern: {([0-9a-zA-Z]*)}
financial Gets Financials For A D-U-N-S number. Invokes the Financial Highlights API duns string up to 15 chars. Pattern: {([0-9a-zA-Z]*)}
familytree Gets Corporate Linkage / Family Tree For A D-U-N-S number. Invokes the Corporate Linkage - Enhanced API duns string up to 15 chars. Pattern: {([0-9a-zA-Z]*)}
firmographic Gets Firmographic Information For A D-U-N-S number. Invokes the Detailed Company Profile - Standard API duns string up to 15 chars. Pattern: {([0-9a-zA-Z]*)}
premfirmographic Gets Firmographic Information For A D-U-N-S number. Invokes the Detailed Company Profile - Premium API duns string up to 15 chars. Pattern: {([0-9a-zA-Z]*)}
findContacts Gets A List Of Contacts For A D-U-N-S number. Invokes the FindContact API duns string up to 15 chars. Pattern: {([0-9a-zA-Z]*)}
news Gets News and Social Media Information for A D-U-N-S number. Invokes the News and Media Service API duns string up to 15 chars. Pattern: {([0-9a-zA-Z]*)}
litefirmographic Gets Basic Company Information A D-U-N-S number duns string up to 15 chars. Pattern: {([0-9a-zA-Z]*)}
refreshcheck Checks when a D-U-N-S number was last updated by D&B. duns string up to 15 chars. Pattern: {([0-9a-zA-Z]*)}
industry Get Research Information on a Hoovers Industry Code (HIC) Invokes the Research - Industry Code API {IndustryTypeCode}-{IndustryCodeTypeCode}
Where IndustryCodeTypeCode must be 25838 (representing the Hoovers Industry Type Code)

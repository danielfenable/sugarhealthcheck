D&B Bulk Import API Overview

Bulk Imports D&B Objects -- Currently supporting Accounts module alone

Request Arguments

Name Type Description Required
bulkdata JSON Array List of objects to be imported True

Request Example

{
"bulkdata":
[
    {
        "name": "Account_1", "duns_num": "1"
    },
    {
        "name": "Account_2", "duns_num": "2"
    }
]
}

Response

The response will contain a json object reporting count of objects successfully imported and count of objects not imported due to duplicates present in the system

Response Example

{
    "importSuccess": 2,
    "duplicates: 0
}

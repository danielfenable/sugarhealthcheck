D&B API -- Post Requests Overview

Interfaces with D&B Direct 2.0 API.
Key to most of the API calls is a D-U-N-S number
A D-U-N-S Number is a unique 9-digit number identifying a business on which a D&B Credit Report has been generated; and once assigned, is neither reused nor assigned to another business.

Query Parameters

Param Description
qtype Query Type: Name of D&B Operation To Invoke Below listed is a table of qtypes. Each of these qtypes represent the different D&B Operations / API's that can be invoked
qdata JSON Object to be POSTed to the D&B operation
